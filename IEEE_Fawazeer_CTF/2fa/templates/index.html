Welcome to Web Development

Web pages are made up of multiple technologies working together behind the scenes to create a functional and visually appealing experience.

  • HTML (HyperText Markup Language): HTML provides the basic structure of a webpage. It tells the browser what elements exist, such as images, text, and buttons.

    Example: <img src="image.jpg" alt="Example Image"> (places an image)

    You can view the **HTML source code** by right-clicking on the page and selecting View Page Source.
  • CSS (Cascading Style Sheets): CSS enhances the appearance of a webpage by adding colors, fonts, spacing, and layout styles to HTML elements.

    Example: body { background-color: lightblue; } (changes background color)

    You can inspect **CSS styles** by right-clicking the page and selecting Inspect (or pressing F12).
  • JavaScript (JS): JavaScript adds interactivity to a webpage by responding to user actions. It enables pop-ups, animations, form validation, and more.

    Example: Clicking the button below will trigger an alert.



    You can view **JavaScript code** by inspecting the page and searching for `<script>` tags.
